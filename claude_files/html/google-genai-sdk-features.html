Implementation Examples

Live API ‚Äî Real-time bidirectional audio/video ‚ñ∂

WebSocket-based, enables voice conversations with Gemini while it generates:

from google import genai
from google.genai import types

client = genai.Client(api_key=API_KEY)

# Start live session
async with client.aio.live.connect(
    model="gemini-2.5-flash-native-audio-preview-12-2025"
) as session:
    # Stream audio input
    await session.send_realtime_input(audio_chunk)

    # Get streaming response
    async for response in session.start_stream():
        if response.audio:
            play_audio(response.audio)
        if response.text:
            print(response.text)

Model: gemini-2.5-flash-native-audio-preview-12-2025

Streaming Responses ‚Äî Show progress immediately ‚ñ∂

Instead of waiting 5-10s for full response, stream tokens as they arrive:

# Current (blocking)
response = await client.aio.models.generate_content(...)

# Streaming (progressive)
async for chunk in client.aio.models.generate_content_stream(
    model="gemini-2.0-flash",
    contents=prompt
):
    if chunk.text:
        yield chunk.text  # Show to user immediately

User sees first token in ~200ms instead of waiting 5-10s

Context Caching ‚Äî 75% cost reduction on repeated content ‚ñ∂

Cache system prompts, reference images, or any repeated context:

from google.genai import types

# Create cache (one-time)
cache = await client.aio.caches.create(
    model="gemini-2.0-flash",
    contents=[system_prompt, reference_images],
    config=types.CreateCachedContentConfig(
        ttl="3600s",  # 1 hour
        display_name="pageant-system-context"
    )
)

# Use cache (75% cheaper for cached tokens)
response = await client.aio.models.generate_content(
    model="gemini-2.0-flash",
    contents=user_prompt,
    config=types.GenerateContentConfig(
        cached_content=cache.name
    )
)

Minimum 32K tokens required to cache. Charged at 25% of input rate.

Code Execution ‚Äî Gemini runs Python in sandbox ‚ñ∂

Let Gemini write and execute Python code to analyze images:

from google.genai import types

response = await client.aio.models.generate_content(
    model="gemini-2.0-flash",
    contents=[
        image_part,
        "Analyze this image's color histogram using matplotlib"
    ],
    config=types.GenerateContentConfig(
        tools=[types.Tool(code_execution=types.CodeExecution())]
    )
)

# Response includes:
# - Generated Python code
# - Execution output
# - Matplotlib charts as images

Available libraries: numpy, pandas, matplotlib. Max 30s execution, 5 iterations.

Embeddings ‚Äî Semantic similarity search ‚ñ∂

Generate vectors for prompts/descriptions to enable similarity search:

result = await client.aio.models.embed_content(
    model="text-embedding-004",
    contents=["sunset over mountains", "beach at dawn"],
    config=types.EmbedContentConfig(
        output_dimensionality=256  # Smaller = faster search
    )
)

# result.embeddings = [[0.1, 0.2, ...], [0.3, 0.4, ...]]

Structured Output ‚Äî Force JSON schema compliance ‚ñ∂

Guarantee response matches your schema (better than regex parsing):

from pydantic import BaseModel

class DesignAnalysis(BaseModel):
    colors: list[str]
    mood: str
    composition: str
    score: int

response = await client.aio.models.generate_content(
    model="gemini-2.0-flash",
    contents=[image_part, "Analyze this design"],
    config=types.GenerateContentConfig(
        response_mime_type="application/json",
        response_schema=DesignAnalysis.model_json_schema()
    )
)

# Guaranteed to parse as DesignAnalysis

Eliminates "Failed to parse annotation JSON" errors in your annotate_design_dimensions

def save_to_favorites(image_id: str, tags: list[str]):
    """Save an image to favorites with tags."""

def regenerate_with_variations(prompt: str, count: int):

response = await client.aio.models.generate_content(
    model="gemini-2.0-flash",

AFC (Automatic Function Calling) enabled by default. Max 10 calls per request.
